単振動の速度・加速度

9章　単振動 — x, v, a の位相関係

← ホームへ戻る

x = A cos(ωt+φ)　v = −Aω sin(ωt+φ)　a = −ω²x

v は x より π/2 進み、a は x と反対向き（位相 π）です。

パラメータを設定して「スタート」

値の連動

ω = √(k/m)。最大速度 vmax = Aω、最大加速度 amax = Aω² です。

角振動数

ω = √(k/m) = — rad/s

T = — s

変位 x(t)

x = A cos(ωt + φ)

→ x = — m

速度 v(t)

v = dx/dt = −Aω sin(ωt + φ)

→ v = — m/s

vmax = Aω = — m/s

加速度 a(t)

a = d²x/dt² = −Aω² cos(ωt + φ) = −ω²x

→ a = — m/s²

amax = Aω² = — m/s²
