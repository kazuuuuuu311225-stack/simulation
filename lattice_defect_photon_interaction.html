格子欠陥と電子励起

物性 · 蛍光・燐光 — 欠陥 → フォノン散乱 → 寿命変化

← メニューに戻る
③ 欠陥 N · A→ ① 振動が乱れる→ ② 寿命が短くなる→ ④ 光が早く消える

①

1 次元格子＋欠陥

空孔 · 欠陥 0 個

格子とフォノン。欠陥の周りだけ位相 δi が乱れ、揺れ方が不揃いになります（フォノン散乱）。

xi(t) = A sin(ωt + iφ + δi)

②

電子励起（S₀ / S₁ / T₁）

散乱が強いほど寿命が短い — 自動遷移

電子の寿命。N と A が大きいほど S₁→S₀（蛍光）・T₁→S₀（燐光）が速くなります。

③

欠陥の設定

種類 · 個数 N · 振幅 A

欠陥の種類
欠陥数 N 2
振幅 A 0.50
蛍光寿命 τ_f —
燐光寿命 τ_p —

τ_f = τ_f0 / (1 + k₁N + k₂A)　　τ_p = τ_p0 / (1 + k₃N + k₄A)
τ_f0=1.0 s, τ_p0=3.0 s, k₁=0.3, k₂=0.5, k₃=0.2, k₄=0.4

④

発光強度の時間減衰

I(t) = exp(−t/τ) — N と A で τ が変化

グラフ。欠陥が多い・振動が強いほど、黄（蛍光）・青緑（燐光）の曲線が急に下がります。

ポイント

  • 格子欠陥は原子の並びの乱れ。
  • 欠陥があると格子振動が乱れ、フォノン散乱が増える。
  • フォノン散乱が増えると電子励起状態の寿命が短くなる。
  • 欠陥が多いほど蛍光寿命が短くなる。
  • 欠陥が多いほど燐光の長寿命が短くなる。
