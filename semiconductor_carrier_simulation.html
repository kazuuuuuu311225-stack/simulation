操作パネル

見方： 青丸 = 電子 e⁻（負電荷）　 赤丸 = 正孔 h⁺（電子が抜けた穴）。
電界をかけると e⁻ は E と逆、h⁺ は E と同じ方向 に動きます（正孔は「逆方向に動く」ように見える）。
① 励起 → e⁻ が VB→CB、h⁺ が残る
② 電界 ON → ドリフト（矢印の方向）
③ pn接合 → 境界で再結合・空乏層
④ LED → 再結合で光

vd = μE　　J = −qD Δn/Δx　　λ ≈ 1240 / Eg (nm)

まず「キャリアの動きを見る」で少ない粒だけゆっくり確認するのがおすすめです。

材料とドーピングを設定し、各ボタンで現象を確認してください。

上段 — エネルギーバンド（CB / Eg / VB）

● e⁻ 伝導帯 CB　● h⁺ 価電子帯 VB　点線 = 同じ位置の e⁻ と h⁺ の対応

下段 — n型 / 空乏層 / p型 とキャリアの動き

左が n型（e⁻ 多数）右が p型（h⁺ 多数）— 各粒の青矢印赤矢印が動く方向
