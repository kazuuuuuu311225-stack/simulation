衝突方向 n に沿って1次元弾性衝突し、接線成分はそのまま。合成して2Dの v′ を求めます。

① 衝突方向 n（球心→球心）

n = (A − B) / |A − B|

→ nx = —、ny = —

② 速度の分解

vn = (v·n) n　　vt = v − vn

A: vA,n = — m/s

B: vB,n = — m/s

③ 法線方向の弾性衝突

v′A,n = (mA−mB)/(mA+mB) · vA,n + 2mB/(mA+mB) · vB,n

v′B,n = 2mA/(mA+mB) · vA,n + (mB−mA)/(mA+mB) · vB,n

→ v′A,n = —、v′B,n = — m/s

④ 2D への再合成

v′ = v′t + v′n n

v′A = (—, —) m/s

v′B = (—, —) m/s

⑤ 運動量保存のチェック

pbefore = mA vA

→ (—, —) kg·m/s

pafter = mA v′A + mB v′B

→ (—, —) kg·m/s

誤差 |Δp| = — kg·m/s

⑥ 運動エネルギー保存

K = ½ m v²　　Ktotal = KA + KB

衝突前 K = — J

衝突後 K = — J

誤差 |ΔK| = — J
