値の連動

復元力 F = −kx より m d²x/dt² = −kx。解は x(t) = A cos(ωt + φ) です。

パラメータ

k = — N/m

m = — kg

φ = — rad

角振動数・周期

ω = √(k/m)

→ ω = — rad/s

T = 2π / ω

→ T = — s

現在の変位

x(t) = A cos(ωt + φ)

t = — s

→ x = — m
