フィックの法則 — 拡散

EX章 · 拡散とフィックの法則

← メニューに戻る

拡散とは、分子がバラバラに動く（ブラウン運動）うえに、 濃度が高い方から低い方へ物質が広がっていく現象です。 生物では O₂ や CO₂ の出入り、細胞膜を越える物質の移動などで重要です。

物理の視点では、分子は温度に応じて常に不規則に動いており（熱運動）、 1 粒ずつ見ると左右どちらへも行き来します。しかし多数の分子を平均すると、 高濃度側から低濃度側へ net（正味）の移動が生じます。これを式にしたのがフィックの法則です。 例えばインクを水に落とす、部屋で香水が広がる、気体が混ざるなども同じしくみで、 温度を上げると分子の速さが増え D が大きくなり、広がりは速くなります。

左：高濃度→ 分子がランダムに動く→ 右へ広がる→ 全体が均一（平衡）

この画面の見方

  • 左の図：1 粒ずつ分子を追った「ミクロ」の様子。シアン＝左側由来、オレンジ＝右側由来。
  • 右のグラフ：左の粒子数を x 方向に数えた濃度 C（青太線）。点線は同じ D から求めた第2法則の予測で、両者が連動します。
  • 黄色の矢印 J：第1法則で求めた拡散の向きと強さ（勾配が大きいほど J も大きい）。

フィックの法則（高校で使う形）

第1法則 — どれだけ・どちらへ流れるか J = −D × (∂C/∂x) J … 拡散流束（単位面積あたりの物質の流れ）／ D … 拡散係数（広がりやすさ）／ ∂C/∂x … 濃度の「傾き」（勾配）。マイナスは高濃度→低濃度向きを表す。
第2法則 — 濃度が時間とともにどう変わるか ∂C/∂t = D × (∂²C/∂x²) 左が高く右が低いと、左の濃度は下がり右の濃度は上がる → 右グラフの段差がなだらかになる。D や温度を上げると、この変化は速くなります。
操作のコツ：まず「開始」→ 速すぎる場合は「再生速度」を下げる → D を上げると広がりは速くなります → 勾配・J が 0 に近づくと「平衡」です。
