W = F × d × cosθ　　Fx = F cosθ

物体は x 方向に動きます。F–d グラフの面積が仕事 W に対応します（縦軸は Fx = F cosθ）。 θ = 90° のとき cosθ = 0 で仕事は 0。θ > 90° では Fx が負になり、+x 方向への移動で負の仕事になります。

スタートで運動を開始

力 F が角度 θ で働き、物体が x 方向に d だけ移動するとき、仕事 W = Fd cosθ です。

① 力 F と角度 θ

F = 40 N、θ = 0°

cosθ = 1.00

② x 方向の力 Fx

Fx = F cosθ

→ Fx = 40.0 N

③ 移動距離 d

d = x − x₀

→ d = 0.00 m

④ 仕事 W

W = F × d × cosθ = Fx × d

→ W = 0.0 J

⑤ F–d グラフ

面積 = 仕事 W（縦軸 Fx、横軸 d）

面積 W = 0.0 J

→ F　 — Fx　 — d　 → v
