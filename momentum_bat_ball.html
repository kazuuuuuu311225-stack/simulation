バットの法線 n に対する入射角 θin と反射角 θout（|θout| = |θin|）から v′ を求め、運動量を計算します。

① 入射角 θin（法線 n に対する角）

θin = ∠(v, n)

→ θin = —°

② 衝突前の速度成分

vx = v cos φ　　vy = v sin φ

→ vx = — m/s、vy = — m/s

③ 反射角 θout

|θout| = |θin|（法線の反対側）

→ θout = —°

④ 衝突後の速度成分

v′ = v′t t + v′n n　（v′n = −vn）

→ v′x = — m/s、v′y = — m/s

⑤ 運動量

pbefore = m v

→ px = —、py = — kg·m/s

pafter = m v′

→ p′x = —、p′y = — kg·m/s

Δp = pafter − pbefore

→ Δpx = —、Δpy = — kg·m/s

● p（衝突前）　 ● p′（衝突後）　 ● Δp — 衝突後に運動量ベクトルを表示
